Scavenger Hunt

Scavenger Hunt is a 3D first person game. Where the objective is to locate items that are randomly placed around the map. In the map you will also find portals that will lead you to puzzle rooms which award the player with a bonus item upon successful completion of the puzzle. What makes Scavenger Hunt interesting is it is also a procedurally generated game meaning the map generated will almost never be the same twice. This game was created using Unity.

My Role:

I was in charge of developing the algorithm that would generate the game map. The map uses perlin noise to create waves which determines which areas are hills and flat lands.
Another task I was assigned was generating the different biomes.In order to achieve this I used the map heights as a threshold in conjuction with perlin noise to determine the different biomes.
Another major task I took lead in was developing a random item placement.
